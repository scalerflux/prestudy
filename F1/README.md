\# 4-Bit Ripple Carry Adder Project



This project implements a structural 4-bit Ripple Carry Adder in Logisim-Evolution using custom 1-bit Full Adder subcircuits.



\## 🛠️ Features

\- \*\*Inputs:\*\* 4-bit DIP switches for Operand A and Operand B.

\- \*\*Modularity:\*\* 4 cascaded 1-bit Full Adder blocks sharing a ripple carry chain.

\- \*\*Output Display:\*\* A 4-bit data bus splitter connected directly to a Hex Digit Display decoder.
