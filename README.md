# Prestudy — Digital Logic in Logisim-Evolution

A hands-on study of digital logic circuits built from the ground up in **Logisim-Evolution**, progressing from basic arithmetic to transistor-level gate design and decoders.

## 📁 Project Structure

### F1 — Arithmetic Circuits
- **4-Bit Ripple Carry Adder** — Structural design using cascaded 1-bit Full Adder subcircuits with hex display output.

### F3 — Gate & Decoder Implementations
- **OR** — Transistor-level OR gate (PMOS/NMOS)
- **NAND** — Static CMOS NAND gate
- **XOR** — NAND-only XOR implementation (16 transistors)
- **XNOR** — XNOR gate circuit
- **Decoder** — Binary decoder with subcircuit demo
